Welcome to the Nubase blog! This is where we'll share updates, tutorials, and announcements about the Nubase framework.

## What is Nubase?

Nubase is a highly-opinionated full-stack framework for creating internal tools and business applications. It's configuration-based, meaning you define your schemas and the framework generates the UI, routing, and API endpoints for you.
